# Data Visualization Portfolio

A comprehensive collection of data visualization projects showcasing various analytical insights and visual representations.

## Projects

### 1. Sales Analytics Dashboard
**Location:** `project_1_sales_analytics/`

**Advanced sales analytics** with comprehensive revenue analysis, category and region performance, temporal patterns, and KPI tracking. Features **6 complex, multi-dimensional visualizations** including revenue trends, category-region heatmaps, revenue-orders correlation, and statistical summaries.

**Files:**
- `sales_analytics_dashboard.py` - Advanced analytics script
- `sales_dashboard.ipynb` - Jupyter notebook (optional)
- `dataset_sales.csv` - Sales dataset (365 days, 5 categories, 5 regions)
- `exports/` - **6 comprehensive visualizations:**
  - `1_comprehensive_revenue_trends.png` - Multi-panel revenue trends dashboard
  - `2_category_region_analysis.png` - Category and region performance
  - `3_revenue_orders_analysis.png` - Revenue vs orders correlation
  - `4_temporal_patterns_analysis.png` - Temporal patterns and seasonality
  - `5_performance_metrics_kpis.png` - Performance metrics and KPIs
  - `6_comprehensive_statistical_summary.png` - Executive summary dashboard

### 5. Global Unemployment Analytics
**Location:** `project_5_global_unemployment_analytics/`
